# Birth Rates and Life Expectancy
Simple R snippet visualising the link between birth rates and life expectancy on a purely descriptive basis.

Further information: [http://www.neurotroph.de/2015/12/birth-rates-and-life-expectancy/](http://www.neurotroph.de/2015/12/birth-rates-and-life-expectancy/)

Data for analysis is not included and can be downloaded from the [World Bank](http://data.worldbank.org). The following indicators were used:

* Life Expectancy: sp.dyn.le00.in
* Birth Rates: sp.dyn.cbrt.in
* Gross Domestic Product: ny.gdp.pcap.pp.kd
